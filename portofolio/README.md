# Rahmat Haikal - Professional Portfolio

Professional single-page portfolio website for Rahmat Haikal, built with modern web technologies.

## Features
- **Responsive Design**: Adapts seamlessly to Mobile, Tablet, and Desktop screens.
- **Modern UI & Glassmorphism**: Premium aesthetic with blurred backgrounds and sleek borders.
- **Dark Mode**: Integrated toggle for Light and Dark themes.
- **Animations**: Typing effects, AOS (Animate On Scroll), floating elements, and a Canvas Particle background.
- **Single Page Architecture**: Smooth scrolling between sections (Hero, About, Experience, Skills, Projects, Contact).
- **SEO Ready**: Semantic HTML and meta tags configured.

## Structure
- `index.html`: Main portfolio page containing all sections.
- `css/`: Stylesheets (`style.css`, `responsive.css`, `animation.css`).
- `js/`: JavaScript logic (`main.js`, `particles.js`, `typing.js`, `darkmode.js`).
- `assets/` & `img/`: Directories for storing images, icons, and CV files (Please add your actual images here).

## Setup Instructions
1. Replace `assets/CV_Rahmat_Haikal.pdf` with your actual CV file.
2. Replace placeholder images in `index.html` with your actual photos and project screenshots.
3. Open `index.html` in your browser to view the portfolio.

## Technologies Used
- HTML5
- Vanilla CSS3 (Custom Variables, Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome 6
- Google Fonts (Poppins)
- AOS Animation Library
